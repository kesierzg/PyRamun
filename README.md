PyRamun - a Python game inspired by first generation Pokémon games

Thank you GameFreak for making my childhood amazing

Name credit goes to my grandma who misspelled Pokémon

This project uses PyGame library.
https://www.pygame.org/docs/

Used font is Dogica-Pixel by Roberto Mocci
https://www.dafont.com/dogica.font

Graphics and sounds are all original creations of mine
