## PyRamun - a Python game inspired by first generation Pokémon games

Thank you Game Freak for making my childhood amazing

Name credit goes to my grandma who misspelled Pokémon

## HOW TO RUN

### 1. Clone the repo or download desired version from releases

### 2. Install Python and dependecnies

Make sure you have Python 3 installed.

Install required packages:

pip install -r requirements.txt

### 2. Run the game

python main.py

Final release will include Windows and macOS packages
